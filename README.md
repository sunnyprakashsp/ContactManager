# 📇 Contact Manager Application

A full-stack **Contact Manager** web application built using **Spring Boot** that allows users to securely manage their contacts with authentication, image uploads, and email features.

---

## 🚀 Features

- 🔐 Secure authentication using **Spring Security**
- 🔑 OAuth2 login with **Google** and **GitHub**
- 👤 User registration and login
- 📇 Add, update, delete, and view contacts
- 🖼️ Contact image upload using **Cloudinary**
- 📧 Email support using **SMTP (Gmail)**
- 🔍 Pagination and search functionality
- 🛡️ Role-based access control
- 📦 Clean MVC architecture

---

## 🛠️ Tech Stack

### Backend
- Java  
- Spring Boot  
- Spring Security  
- Spring Data JPA (Hibernate)

### Database
- MySQL

### Frontend
- HTML  
- CSS  
- JavaScript

- OAuth2 (Google & GitHub)
- Cloudinary (Image Storage)
- Maven
